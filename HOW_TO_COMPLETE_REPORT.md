# How to Complete Your Report

## Quick Guide - 5 Steps

### Step 1: Add Your Group Information

Open `PROJECT_REPORT.md` and find the **Cover Page** section. Replace:

```
- [INSERT GROUP MEMBER 1 NAME] ([INSERT STUDENT ID])
- [INSERT GROUP MEMBER 2 NAME] ([INSERT STUDENT ID])
- [INSERT GROUP MEMBER 3 NAME] ([INSERT STUDENT ID])
```

With your actual names and student IDs.

---

### Step 2: Add Your YouTube Video Link

After you create and upload your video:

1. Find the section: **"3. YouTube Video Link"**
2. Replace: `[INSERT UNLISTED YOUTUBE VIDEO LINK HERE]`
3. With your actual YouTube URL

**Important:** Make sure the video is set to **"Unlisted"** not "Private" or "Public".

---

### Step 3: Fill Out Individual Contribution Report

Find **"Appendix A: Individual Contribution Report"** and for each team member:

**Replace the template with actual details:**

```
### Team Member 1: [INSERT NAME]

**Responsibilities:**
- Created HTML structure and layout for all screens
- Implemented form validation and error handling
- Wrote JavaScript state management

**Code Contributions:**
- index.html - All HTML structure
- app.js - Form validation functions, state management
- styles.css - N/A

**Report Contributions:**
- Wrote Section 1.1 (Overview)

**Other Contributions:**
- Initial project planning, design decisions
```

**Be specific and honest about who did what!**

---

### Step 4: Fill Out Meeting Notes

Find **"Appendix B: Meeting Notes"** and:

**For each meeting:**
1. Replace `[INSERT DATE]` with actual date
2. Replace `[LIST ALL NAMES]` with actual attendee names
3. Fill in actual decisions made
4. List real action items assigned
5. Replace `[INSERT TIME]` with duration

**Example:**
```
### Meeting 1: November 15, 2024

**Attendees:** John Doe, Jane Smith, Bob Johnson

**Decisions Made:**
- Selected HTML/CSS/JavaScript (no frameworks)
- Chose dark theme for modern feel
- Agreed on 3-step booking process

**Action Items:**
- John: Create initial HTML structure
- Jane: Design color scheme
- Bob: Research price estimation

**Duration:** 1.5 hours
```

**If you had fewer/more meetings:** 
- Delete extra meeting sections OR
- Add more meeting sections using the same format

---

### Step 5: Convert to PDF and Create ZIP

**Option A: Using Markdown to PDF (Recommended)**

1. Open `PROJECT_REPORT.md` in a markdown editor or VS Code
2. Use a Markdown to PDF converter (like "Markdown PDF" extension in VS Code)
3. Export as PDF
4. Name it: `[YourSlackChannelName]_Report.pdf`

**Option B: Using Google Docs/Word**

1. Copy the content from `PROJECT_REPORT.md`
2. Paste into Google Docs or Microsoft Word
3. Format nicely (headings, spacing, etc.)
4. Export as PDF
5. Name it: `[YourSlackChannelName]_Report.pdf`

**Create ZIP File:**

1. Create a new folder
2. Put these files in it:
   - Your PDF report
   - index.html
   - styles.css
   - app.js
   - README.md
3. ZIP the folder
4. Name it: `[YourSlackChannelName].zip`

## Pro Tips

1. **Be Specific in Contributions:** Instead of "worked on code," say "implemented form validation system in app.js"
2. **Show Collaboration:** Mention group decisions and code reviews
3. **Meeting Notes Matter:** Detailed notes show your process and collaboration
4. **Proofread:** Check for spelling, grammar, and formatting before converting to PDF
5. **Test Your ZIP:** Extract it and make sure all files are there and the prototype works
