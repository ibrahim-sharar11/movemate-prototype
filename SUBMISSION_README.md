# Submission Instructions

## Files to Submit

Create a ZIP file containing:

1. `[YourSlackChannelName]_Report.pdf` - Your completed report (convert PROJECT_REPORT.md to PDF)
2. `index.html` - Main HTML file
3. `styles.css` - All CSS styling
4. `app.js` - All JavaScript functionality
5. `README.md` - Technical documentation

## Before Submitting

1. **Fill in PROJECT_REPORT.md:**
   - Add group member names and student IDs (Cover Page)
   - Fill out Appendix A (Individual Contributions)
   - Fill out Appendix B (Meeting Notes)
   - Add your YouTube video link (Section 3)

2. **Create YouTube Video:**
   - Follow VIDEO_SCRIPT.md for guidance
   - Record 5-minute demonstration
   - Upload as unlisted
   - Add link to report

3. **Convert Report to PDF:**
   - Export PROJECT_REPORT.md as PDF
   - Name it: `[YourSlackChannelName]_Report.pdf`

4. **Create ZIP:**
   - Package all 5 files above
   - Name it: `[YourSlackChannelName].zip`

5. **Submit to Canvas** before December 1, 11:59 PM
