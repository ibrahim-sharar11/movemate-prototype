## üé• Recording Tips

1. **Start Recording:**
   - Have prototype already open
   - Start at landing screen
   - Clear browser console (F12, right-click, Clear console)

2. **During Recording:**
   - Speak clearly and at moderate pace
   - Click buttons deliberately (not too fast)
   - Pause briefly on important features
   - If you make a mistake, pause and restart that section

3. **Screen Setup:**
   - Fullscreen browser window
   - 1920x1080 or similar resolution
   - Good contrast (dark theme shows well)
   - Zoom at 100% (not zoomed in/out)

4. **Audio:**
   - Use good microphone if narrating
   - Record in quiet environment
   - Or add text annotations instead

5. **Editing (if needed):**
   - Trim dead air at start/end
   - Cut out mistakes
   - Add text overlays if not narrating
   - Ensure it's exactly 5 minutes or under
