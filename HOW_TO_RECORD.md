# üé• How to Record Your Video

## ‚úÖ BEST OPTION FOR MAC: QuickTime Player

**Why:** Built-in, free, simple, good quality

### Steps:

1. **Open QuickTime Player**
   - Search "QuickTime" in Spotlight (Cmd+Space)
   - Or find in Applications

2. **Start Recording**
   - File ‚Üí New Screen Recording
   - Click the red record button

3. **Select Recording Area**
   - Click and drag to select your browser window
   - Or click "Record Entire Screen"

4. **Record**
   - Click "Record" button
   - Do your demo (5 minutes)
   - Click stop button in menu bar (top right)

5. **Save**
   - File ‚Üí Save
   - Name it: `movemate-demo.mp4`
   - Save to Desktop or Documents

6. **Optional: Trim**
   - QuickTime can trim start/end if needed
   - Edit ‚Üí Trim

## Alternative Options:

### macOS Built-in (Cmd+Shift+5)
- Press Cmd+Shift+5
- Select "Record Entire Screen" or "Record Selected Portion"
- Click Record
- Stop from menu bar

### Loom (Online - Free)
1. Go to loom.com
2. Install browser extension
3. Click Loom icon
4. Select screen/browser tab
5. Record and it uploads automatically
6. Get shareable link

### OBS Studio (Free, Advanced)
- Download from obsproject.com
- More features but more setup
- Good if you want to add overlays/effects

## üí° Recording Tips:

### Before Recording:
- [ ] Close unnecessary apps/tabs
- [ ] Clean your desktop
- [ ] Resize browser to good size (not too small/large)
- [ ] Have prototype ready (open in browser)
- [ ] Test audio (if narrating)
- [ ] Clear browser console (F12 ‚Üí right-click ‚Üí Clear)

### During Recording:
- Speak clearly and at moderate pace
- Click buttons deliberately (not too fast)
- Pause briefly on important features
- If you mess up, pause and restart that section

### After Recording:
- Watch it back to make sure it's clear
- Trim any dead air at start/end
- Should be ~5 minutes or less

## üìê Recommended Setup:

**Screen Size:**
- Record at 1920x1080 if possible
- Or your screen's native resolution
- Make browser window large but not fullscreen

**Browser:**
- Use Chrome or Safari
- Fullscreen the prototype window
- Zoom at 100% (not zoomed in/out)

**Quality:**
- QuickTime records in good quality by default
- No need to adjust settings usually
